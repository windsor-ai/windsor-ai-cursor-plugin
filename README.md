# Windsor.ai Cursor Plugin

Cursor plugin for querying marketing and business data from 350+ sources with [Windsor.ai](https://windsor.ai).

## What's included

- **MCP Server** — Connects to Windsor.ai's API via the Model Context Protocol. Exposes tools for discovering connectors, exploring fields, and querying data.
- **Skills**
  - `/marketing-analyst` — Analyze ad spend, conversions, ROAS, and trends across channels.
  - `/data-explorer` — Discover available connectors, fields, and data sources.
- **Agent** — Marketing data analyst that answers business questions grounded in real data.
- **Rules** — Best practices for the Windsor.ai query workflow (connectors → options → fields → data).

## Setup

1. Install the plugin from the Cursor Marketplace.
2. Get a Windsor.ai API key at [onboard.windsor.ai](https://onboard.windsor.ai).
3. When prompted in chat, provide your API key.

## Supported data sources

Windsor.ai connects to 350+ platforms including Google Ads, Facebook Ads, Google Analytics 4, Salesforce, LinkedIn Ads, TikTok Ads, and more.
